# 📊 로또플레이(Lottoplay) 서비스 및 조합 시스템 상세 분석 보고서

본 문서는 `lottoplay.co.kr` 사이트의 구조, 제공 데이터, 그리고 골드회원 전용 번호 생성 알고리즘을 정밀 분석한 결과입니다.

---

## 1. 서비스 개요 및 구조
- **도메인**: `https://www.lottoplay.co.kr`
- **핵심 가치**: 17년 누적 데이터를 기반으로 한 통계적 번호 추출 및 필터링 서비스 제공.
- **기술적 특성**: 
    - **Encoding**: EUC-KR (한글 처리를 위해 `iconv-lite` 필수).
    - **Rendering**: 서버 사이드 렌더링(SSR). 데이터가 HTML에 직접 노출되어 크롤링 효율이 높음.
    - **Device Handling**: 모바일 접속 시 리다이렉트(`m.lottoplay.co.kr`) 정책 운영.

---

## 2. 골드회원 번호 조합 시스템 분석
로또플레이의 유료 핵심 모델인 '골드시스템'은 단순 랜덤이 아닌 **통계적 압축 기법**을 사용합니다.

### 2.1 핵심 필터링 로직
1.  **합계 범위 (Sum Filter)**: 6개 번호의 총합이 정규 분포의 90% 이상을 차지하는 **100~170** 사이의 조합만 추출.
2.  **홀짝 비율 (Odd/Even Filter)**: 극단적인 경우(0:6, 6:0)를 제외한 **2:4, 3:3, 4:2** 비율 적용.
3.  **저고 비율 (Low/High Filter)**: 전체 번호대(1~45)를 절반으로 나누어(1~22, 23~45) 균형 있는 배분 유도.
4.  **연번 필터 (Consecutive Filter)**: 3개 이상의 연속 번호(예: 1,2,3)가 포함된 저확률 조합 제거.

### 2.2 전문 조합기 구성
- **필터링 조합기**: 과거 당첨 데이터를 기반으로 출현 가능성이 낮은 패턴 제거.
- **퍼펙트 조합기**: 15~20단계의 세부 필터링을 거친 최적의 압축 조합.
- **고정수/제외수 시스템**: 사용자 지정 번호를 반영한 가중치 적용 시스템.

## 3. 데이터 라이브러리(Library) 구성 내역
현재 프로젝트에서 수집 완료한 데이터 카테고리입니다.

| 단계 | 주요 항목 | 활용 방안 |
|:---:|:---|:---|
| **Elementary** | 번호별 통계, 출현순위, 총합/홀짝 통계 | 기본 분석 대시보드 및 리포트 구성 |
| **Intermediate** | 이웃수, 연번, 9궁도, 소수, 3/5배수 현황 | 패턴 인식 모델의 입력 데이터(Input) |
| **High Level** | 회귀설, 낙첨값, 가로/세로라인, 궁합 리스트 | AI 기운 분석 엔진의 정밀도 향상 |

## 4. 우리 프로젝트(`lotto-map-korea`) 적용 전략

### 4.1 AI 기운 분석 엔진 고도화
- 로또플레이의 **'낙첨값'**과 **'회귀설'** 데이터를 결합하여, 단순 운세가 아닌 통계적 근거를 가진 번호 추천 서비스 제공.
- 판매점별 **'수동 당첨 비중'**이 높은 곳에 특화된 분석 알고리즘 적용.

### 4.2 실시간 데이터 통합
- **예상 당첨금**: 실시간 판매량 데이터를 가져와 메인 대시보드에 생동감 부여.
- **최근 당첨지**: 로또플레이의 실시간 당첨점 정보를 지오코딩하여 지도 위에 즉시 반영.

### 4.3 마케팅 및 전환(Conversion)
- 로또플레이의 **'사회적 증명(Social Proof)'** 방식(당첨 영수증 인증 등)을 참고하여 사용자 참여형 게시판(Issue #4) 구축 검토.
