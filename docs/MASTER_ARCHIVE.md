# 📚 LottoMap Master Archive: 데이터 및 알고리즘 통합 보고서

## 1. 수집 데이터 자산 (Data Assets)
로또플레이(Lottoplay) 라이브러리 전수 조사를 통해 확보한 RAW 데이터 리스트입니다.

### 1.1 통계 라이브러리 (32개 페이지)
- **Elementary (12개)**: 번호별 빈도, 순위, 홀짝/고저 비율, 미출현 기간 등.
- **Intermediate (12개)**: 이웃수, 연번, 9궁도, 소수, 3/5배수 등 패턴 데이터.
- **High Level (8개)**: 회귀설, 낙첨값, 가로/세로라인, 번호 궁합(Good/Bad) 리스트.

### 1.2 정보 및 지식 베이스 (5개 페이지)
- 로또 용어 사전, 당첨금 수령 가이드, 15가지 필터링 구입 요령 등.

## 2. 경쟁사 알고리즘 분석 결과

### 2.1 로또플레이 '골드 시스템' (Filtering Logic)
- **압축 알고리즘**: 814만 분의 1 확률을 통계적 정규 분포(합계 100~170 등)를 통해 약 1/500 수준으로 압축.
- **사용자 변수**: 고정수와 제외수를 상수로 두는 조건부 확률 모델 사용.

### 2.2 로또리치 '로또랩스' & '꿈해몽' (UX & Psychology)
- **LottoLabs**: 12대 분석 지표를 시각화하여 사용자가 직접 필터를 조절하게 만드는 '분석 효능감' 전략.
- **Dream AI**: 비정형 꿈 키워드를 정형 번호와 매핑 후 랜덤워크(Random Walk) 탐색으로 최종 조합 생성.

## 3. 기술적 구현 성과 (v2.6)
- **ETL 파이프라인**: Node.js(Cheerio)와 Python(Pandas)을 결합한 자동 수집/정제 시스템 구축.
- **지오코딩 최적화**: 220개 명당 데이터의 좌표 캐싱 및 배치 처리 완료.
- **시각화 엔진**: Chart.js를 통합하여 번호별 통계를 직관적인 대시보드로 구현.
