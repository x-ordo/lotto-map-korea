# 📑 로또명당 프로젝트 통합 분석 및 구현 보고서 (v2.6)

본 보고서는 로또 명당 지도 서비스의 경쟁력 강화와 데이터 기반 분석 시스템 구축을 위해 수행된 모든 조사 및 구현 내용을 담고 있습니다.

---

## 1. 경쟁사 분석 및 벤치마킹 결과

### 1.1 로또플레이 (Lottoplay.co.kr)
- **핵심 기술**: 17년 누적 데이터를 활용한 '골드 시스템' 필터링 조합.
- **가져온 요소**: 
    - **통계 라이브러리**: 초급/중급/고급 단계의 32개 분석 페이지 데이터 구조.
    - **번호 조합 공식**: 합계(100~170), 홀짝 비율, 낙첨값 가중치 로직.
    - **실시간 당첨지**: 매주 업데이트되는 1등 배출점 상세 주소 데이터.

### 1.2 로또리치 (Lottorich.co.kr)
- **핵심 전략**: 압독적인 당첨 증명(Social Proof)과 심리적 전환 퍼널.
- **가져온 요소**:
    - **사회적 증명 UI**: 실제 1등 배출 카운터 및 당첨 영수증 인증 방식.
    - **마케팅 트리거**: 꿈 해몽 시스템을 통한 사용자 리텐션 강화.
    - **SEO 최적화**: 롱테일 키워드(꿈해몽, 세금계산기 등) 유입 전략.

---

## 2. 데이터 인프라 및 수집 시스템 (ETL)

### 2.1 다중 소스 수집기 (Scrapers)
- **Node.js + Cheerio**: 로또플레이의 EUC-KR 인코딩 페이지 정밀 파싱.
- **Python + BeautifulSoup/Pandas**: 고성능 통계 분석 및 시각화 차트 생성 엔진.
- **Playwright**: 동적 사이트 구조 분석 및 숨겨진 인터랙션 추적.

### 2.2 지오코딩 최적화 (Optimization)
- **캐시 전략**: `data/geoCache.json`을 통한 API 호출 비용 및 시간 절감.
- **배치 처리**: 10개 단위 병렬 요청으로 수집 속도 10배 향상.
- **알고리즘 튜닝**: 주소 정규화(Normalize)를 통한 검색 매칭률 극대화.

## 3. 핵심 기능 구현 현황

### 3.1 AI 지능형 분석 엔진
- **Luck Index (명당 지수)**: 단순 당첨 횟수가 아닌 지역 판매량 대비 효율을 수치화.
- **꿈 해몽 번호 추출**: 20가지 핵심 길몽 키워드 기반 번호 매핑 및 AI 해석.
- **실시간 기류 분석**: 동행복권 API와 연동된 실시간 당첨 정보 투영.

### 3.2 UI/UX 고도화 (v2.6)
- **탭 시스템**: 지도, 순위, 분석, 꿈해몽, QR로 이어지는 멀티 탭 구조.
- **기기 최적화**: S24 등 최신 기기 레이아웃 겹침 방지(Grid System).
- **외부 연동**: 네이버 지도 길찾기 다중 창 버그 수정 및 전화 걸기 연동.
- **스피또 정보**: 2000/1000/500 모델별 실시간 잔여 1등 매수 시각화.
